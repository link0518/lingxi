根据现有的前端页面完成项目   仿微信和酒馆设计 大部分功能对齐微信 
本系统设计如下：ai伴侣项目 项目名称：灵犀
1.消息页用来和用户聊天 左栏聊天列表 右栏聊天内容 用户可以删除会话 删除从再次从角色卡打开聊天为新对话
2.通讯录 展示所有角色卡信息 加号可以创建角色卡 左栏角色卡列表 点击左栏 展示角色卡详细  支持编辑和删除 角色卡可以上传头像 用到朋友圈和聊天展示头像 
3.朋友圈 展示所有角色卡的朋友圈 用户也可以发布朋友圈 完全仿照微信 点赞评论 
4.我的  展示用户设定 可以修改头像昵称和设定 
5.设置  展示用户设置 可以修改密码和退出登录 依旧是左右栏设计 左栏设置选项 右栏设置内容 设置选项如下：
- 提示词预设 仿照酒馆的预设功能 分块加载 有默认提示词 用户可以添加自定义提示词 依旧膝盖提示词内容 
- api 设置 ：url key model 支持自动获取model列表    
- 管理面板：仅管理员可见 具体选项有  
1.生成邀请码 用户注册需要邀请码 邀请码仅可使用一次 能管理邀请码 复制邀请码 
2.用户管理：展示所有用户信息和查看用户的所有聊天记录 修改用户的密码
3.好感度设定：管理好感度系统 调整好感度算法 好感度的各个阶段 以及每阶段的提示词，ai根据当前好感度选择提示词，对用户的聊天风格 从陌生到亲密 调整不同的回复长度以及语气  
技术栈 react+vite+tailwind 
系统设计为 支持长效记忆和好感度系统 用户登录系统 隔离每个用户的数据  ai回复的提示词为预设+角色卡设计+长效记忆+短期记忆 以及能读到用户的设定 （仿照酒馆设计） 采用sqlite数据库  数据本地+数据库存储 聊天体验接近微信 可以设置调用两种大模型（用户选择） 聊天模型和其他任务模型  聊天模型专注于回复用户的聊天内容 其他任务模型用来处理好感度 记忆等内容（总结用户对话，保存关键记忆等） 方便聊天模型调用 也可以只使用一个聊天模型
